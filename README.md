**Banking Risk Analytics Project**
This project uses Power BI to analyze banking and client data to help banks make better lending decisions.It shows whether a loan applicant is likely to repay or not, helping reduce financial risk.

**Dataset**
The data includes several related tables such as Banking Relationship, Client-Banking, Gender, Investment Advisor, and Period. These contain client details, account information, and loan data.

**Data Cleaning**
Added columns for Engagement Timeframe and Engagement Days to track client duration.
Created Income Band to group clients by income level.
Added Processing Fees based on fee structure.

**Key Metrics**
Total Clients – Number of unique clients
Total Loans – Combined total of all loan types
Total Deposits – Total amount deposited
Total Fees – Charges based on loan and fee structure
Engagement Length – Days clients stayed with the bank

**Dashboards**
Home Dashboard – Overview of all metrics
Loan Analysis – Loan details by client and type
Deposit Analysis – Deposit trends and patterns
Summary Dashboard – Combined KPIs and insights

**Conclusion**
Power BI dashboards help banks understand client behavior, manage risk, and make data-driven loan decisions.
